Execution Phases
1
Phase 1 · Now Mine SP Data for Intent Signals
  • Pull search term reports (last 30–60 days)
  • Identify top converting intent clusters — query signals, not demographics
  • Identify high-click / no-convert terms → add as negatives at originating campaign/ad-group level by match type (auto, broad, phrase, product-targeting)
  • Identify conquest ASINs: competitor ASINs with ≥10 clicks AND ≥1 order (both required — not OR)
Output Intent cluster map · Negative keyword list · Qualifying conquest ASIN list
1½
Phase 1.5 · Before SD Tighten SP — Per-SKU Exact Audit
  • Check ads-change-log for existing exact campaigns per SKU
  • Audit current exact targets vs Phase 1 intent clusters — fill gaps only, no duplicates
  • Add anti-cannibalization negatives (exact negatives on auto, phrase negatives on broad)
SD Launch Gate · All must pass
  • Primary SP discovery/harvest campaigns show ACOS within target range
  • 2 consecutive weekly check-ins at min $50/week aggregate spend
  • SP CVR baseline established per SKU
2
Phase 2 Build SD Targeting Brief
  • Map qualifying intent clusters → SD keyword context targeting
  • Map qualifying conquest ASINs (threshold met in Phase 1) → SD product targeting
  • Define remarketing audience: product page viewers, 30-day window, no conversion
Output SD campaign configuration document with pre-validated targets
3
Phase 3 Launch Sponsored Display
3 Campaigns
  • A — Views remarketing (product page visitors)
  • B — Keyword context (top SP intent clusters)
  • C — Conquest ASIN (threshold-qualified only)
Budget: $20–30/day per campaign
Early Kill Rules · Check First
  • 0 orders after $100 spend → pause
  • ACOS > 60% after $50 spend → pause
Optimization Review Gate
  • Min 50 clicks
  • Min $50 spend
  • Min 1 order
  • If SD ACOS > 2× SP ACOS → flag & reassess
4
Phase 4 · After 30 days SP + SD AMC Descriptive Analysis
  • Query AMC: % of converters with SD exposure vs SP-only
  • Strictly descriptive — correlation only, not causal lift (SD-exposed shoppers self-select)
  • No pseudo-holdout — geography/time comparisons are confounded by seasonality
  • Cadence: bi-weekly runs, not ad-hoc
DSP Planning Gate · Phase 5 trigger
  • Same directional relationship across 2 consecutive bi-weekly runs
  • Minimum 30 converters per run
5
Phase 5 · Follow-on plan DSP — If AMC Justifies It
  • Retarget off-Amazon using Amazon's shopper data
  • Requires Phase 4 gate to pass — separate entry criteria document
  • Budget minimum: ~$10k/mo — separate budget approval required
  • Not detailed here
Known Risks
Monitor throughout execution
  • SP data volume may be too thin for reliable intent clusters — if <200 clicks in 30 days, Phase 1 output will be limited
  • SD keyword context targeting is broad — monitor for impression share from irrelevant categories
  • Price gap ($55.08 vs $8–15 competitors) is structural — SD cannot fix a broken offer; CVR baseline must pass before SD launch
  • AMC signal may be inconclusive if SD runs below ~1,000 impressions/day
  • SD reporting extension to daily report script must be built before launch — not optional
