The diagnosis pivot reframed

The earlier working theory was "bids too low for the laundry category" ($1.50–3.00 winning CPC vs your $0.35–0.85). That's real for the head keyword terms — but it cannot explain a literal zero everywhere:

  • Low bids produce few impressions, not zero.
  • Auto & product-targeting clear cheaply on every other Everneat line — but are also dark for LS.
  • Manually bumping ~10 broad terms to $1.20–2.00 still got nothing.

A universal zero, while sibling ASINs serve on similar structure, fits a specific failure — a stray negative keyword/product exclusion, a routing conflict, a stale/zero target bid, or a structural mismatch we haven't literally diffed. The adversarial review flagged that our own change log has flip-flopped on this ("listing" → "bid floors" → "bids"), so no prior conclusion is trusted.

The plan — diagnose first (free), then decide

Phase 0 runs entirely through the Ads API: no budget change, no bid change, no UI. It's the cheapest and highest-information action available.

1 Structure diff — per campaign type mandatory

Field-by-field compare each LS campaign to a serving sibling of the same type (auto↔auto, keyword↔keyword, product↔product): bid strategy, placement multipliers, dates, budgets, ad-group & target bids, negatives, promoted ASINs, product-ad state, profile, portfolio. "Identical structure" must be proven, not assumed.

2 Negative-conflict audit mandatory

Check every keyword & product-target negative — at campaign and ad-group scope — plus cross-campaign routing negatives, for anything silently wiping LS traffic.

3 Bid-source audit mandatory

Confirm campaign vs ad-group default vs target-level bids interact as expected — no stale or zeroed target bids hidden behind the campaign summary.

4 Placement + impression-share reports supporting only

Corroboration only — we do not draw a root cause from missing report data.

5 Fixed-bid isolate test — the decider

If steps 1–3 come back clean: pause all but one highly-relevant exact keyword, set fixed bidding (removes the dynamic-down confounder), run a ladder $2.00 → $2.50–3.00 for 48–72h each, same $7/day cap. Then repeat on one product target only if the keyword stays zero. This proves whether the ASIN can enter any auction at all.
