All 6 LS Sponsored Products campaigns serve ~0 impressions. Audits to date have found no blocking config/negative/bid-plumbing defect (structure diffed, negatives audited, bids present, buy box held, listing live) — but earlier diagnoses on this have been unstable, so this is "nothing found," not "impossible." The leading hypothesis is an ASIN-relevance / cold-listing limit (5 reviews, low velocity, $1.50–3.00 CPC category) — but this is not yet proven. A live fixed-bid isolate test resolves it:

HARD GATE — 2026-07-25 isolate verdict. If the isolated keyword gets impressions at ≤$2.50 fixed → a bid/auction lever is viable and this listing-led plan is scaled back. If it stays at zero at that ceiling → bid-only recovery is not supported at this ceiling; proceed to listing-side remediation unless further auction testing (higher ceiling / more terms) is approved. (One ceiling failing does not by itself prove relevance is the cause — it rules out cheap bid recovery.) Do not commit P0 spend on listing work until this gate is read.

Bid/budget changes alone are unproven and should not be scaled until the verdict is in. No blanket bid/budget increase is requested here.

Preparatory vs. committed work at the gate: preparatory, reversible work may begin before 07-25 (drafting copy, shooting images, running the Vine eligibility check, confirming inventory). Financial or irreversible commitments — reserving Vine units, publishing listing changes, releasing test budget — are blocked until the verdict is read.

8. Success metrics & kill criteria (concrete, attributable)

Two separate decision cycles so causality isn't blurred:

Short-cycle (ad auction, ~1 week): - Exit-zero threshold: ≥500 impressions AND ≥5 clicks AND ≥$5 spend on the isolate/long-tail over a 7-day window (0→trivial does NOT count). - CTR target uses an internal comparator, not an external benchmark we can't source: LS SP CTR should reach ≥50% of the account's own OS/SR SP CTR (pull both from the same report, same placement mix, same window). Rationale: same account, same ad system, so it's an apples-to-apples floor.

Long-cycle (listing/reviews, ~4–8 weeks): - Reviews ≥20; CVR ≥ +30% vs the pre-intervention 4-week baseline and detail-page-view-rate ≥ +20% vs baseline (directional-but-measured, not "trend up"). - LS-specific stage metrics (auction entry rate, CTR, CVR, CPC, TACOS contribution) — NOT parity with the OS line (different product/demand/economics).

Instrumentation (required): a dated experiment log capturing each intervention (listing edit, Vine date, price/coupon change, each ad test) with pre/post windows, so effects are attributable. Owner: ads team.

Kill criteria (staged by dated intervention, so attribution stays clean): - Ad-side (07-25 + 3 days): isolate still can't clear at ≤$2.50 fixed → no scaling of SP bids; pause and redirect. - Listing-copy checkpoint (4 wks after copy+images publish, baseline = pre-publish): if impressions/CTR haven't moved → copy/relevance is not the lever. - Reviews checkpoint (4 wks after ≥20 reviews land, baseline = pre-Vine): if CVR/impressions haven't moved → reviews weren't the limiter either → pause LS paid search, redirect to OS/SR/DS.
